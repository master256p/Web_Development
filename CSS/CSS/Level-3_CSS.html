2. Position
static || relative || absolute || fixed

i. static

Static - default postion (The top, right, bottom, left, and z-index properties have no effect).
Relative - element is relative to itself. (The top, right, bottom, left, and z-index will work).
absolute - positioned relaive to its closest positioned ancestor. (removed from the flow).
fixed - positioned relative to browser. (removed from flow)
sticky - positioned based on user's scroll position.
